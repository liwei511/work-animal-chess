# 职场斗兽棋 - 中二职场版动物棋

传统斗兽棋改成互联网职场版本，还是那个味道，全是梗，中二拉满。

## 规则对应

传统斗兽棋 8 种动物 → 互联网公司 8 个职级：

| 传统 | 职场版 | 吃法 | 分数 |
|------|--------|------|----|
| 象 | CEO | 吃 CTO-总监-经理-专员-实习生 | 8 |
| 狮 | CTO/CPO | 吃 总监-经理-专员-实习生 | 7 |
| 虎 | 总监 | 吃 经理-专员-实习生 | 6 |
| 狗 | 经理 | 吃 专员-实习生 | 5 |
| 狼 | 高级专员 | 吃 实习生 | 4 |
| 猫 | 专员 | 吃 实习生 | 3 |
| 鼠 | 实习生 | 能吃 CEO！初生牛犊不怕虎 | 1 |
| （新增） | 投资人 | 能吃所有人！ | 9 |

## 特殊规则

**遵循传统斗兽棋规则：**
- 高级可以吃低级，低级只能吃同级更低
- 实习生（鼠）可以吃 CEO（象），神奇吧
- 新增 **投资人**，谁都能吃，没人能吃投资人
- 陷阱 → 需求变更池：任何职位进去，都会被对方吃掉。"需求变了，这个版本砍了"
- 河流 → 跨部门沟：- 只有产品经理（…哦不对我们这里没有，应该是 只有总监以上才能直接跨部门？不对，传统是只有鼠猫狗能过河，改成 只有经理及以上才能跨部门沟通！
- 大本营 → 自己的工位：对方进来就被吃掉。"这是我的工位，你滚"

## 中二台词

每个棋子被吃的时候都有中二台词：

- CEO 被实习生吃："没想到吧，新老板已经在外面找好了，你可以优化了"
- 总监被经理吃："年轻人真能卷，我老了，让位给你"
- 经理被实习生吃："今天这需求我真不想改，你年轻人精力旺你来吧"
- 投资人吃CEO："故事讲的不错，这轮我投了，你下课"
- 实习生吃投资人："我不接受融资，我自己说了算"（rare case

## 棋盘

8x8 = 一共 64 格子，两边分红蓝，对应两个公司对战。

## 胜负

一方杀光另一方就算赢，或者对方没棋走了就算赢。

## 整体中二风格

- 每个棋子头像用emoji就行，先凑活
- 台词全是互联网梗，打完一把笑得肚子疼
- 两个人对战，摸鱼正好玩一把
